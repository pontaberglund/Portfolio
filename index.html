ChipKit Snake

This was a school project for the course IS1200 Computer Hardware Engineering. The task was to create something using the chipKIT Uno32 and a I/O shield with some buttons and an oled screen. Me and my partner Hugo Bachér decided to create a snake game to be played on the oled screen with the built in buttons. This project teached me programming on a lower level using both assembley and C to communicate with the hardware.
